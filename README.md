A fork of DiK's "Store Block - Create your own Trade Station" mod.

Implements multiple Admin-Only Store blocks that can be used for trading. NPC Dummy versions with models from Decorative Pack 1 DLC are included as well for those who want an NPC shop feel.

It allows the same offers as the economy stations and custom items without mod adjustments.
The blocks are filled with offers and materials with every restart, power on or after 20 minutes.

Supported:
- Ore
- Ingot
- Components
- Ammunition
- Tools / Weapons / Bottles
- Consumable Items
- Ships
- custom objects from other mods of the same classes

Shop Setup:
Place your store of choice from this mod while in Creative Mode
Open the store's CustomData
Configure the Sales Options
Turn block off and on and transfer the block to an NPC faction
